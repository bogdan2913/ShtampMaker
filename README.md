# ShtampMaker

Инструмент для управления и простановки штампов на PDF-файлы.

## Что это

Два компонента:

- **Приложение** — управление библиотекой штампов: добавление, категории, избранное, корзина
- **stamp_pdf.py** — простановка штампа на PDF через контекстное меню Проводника Windows

## Установка

**Требования:** Python 3.10+, Windows

```bash
git clone https://github.com/bogdan2913/ShtampMaker.git
cd ShtampMaker
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Структура штампов

Штампы хранятся в папке `stamps/` — каждая подпапка является категорией:

```
stamps/
├── Общие/
│   └── иванов_5x2.png
└── Подписи/
    └── петров_3x1.5.png
```

Имя файла кодирует физический размер: `название_ШxВ.png` (ширина × высота в сантиметрах).  
Добавление только через интерфейс приложения.
